# 游戏记录文章示例

这个目录存放你的游戏生涯记录文章。

## 快速开始

### 1. 使用模板创建新文章

```bash
hexo new game "游戏名称"
```

这会使用 `scaffolds/game.md` 模板创建新文章。

### 2. 填写 Front-matter

```yaml
---
title: "游戏名称"
date: 2024-01-15 14:30:00
game_id: "game-unique-id"  # 同一游戏用相同ID
track: "single"            # single 或 multi
game_type: "rpg"
platform: "PC"
play_time: "50"
clear_date: "2024-02-01"
rating: 9
description: "简短描述"
categories:
  - 单机游戏
tags:
  - RPG
  - 开放世界
photos:
  - /assets/images/games/游戏名/截图1.jpg
  - /assets/images/games/游戏名/截图2.jpg
videos: []
completion_rate: 80
achievements:
  - "成就1"
  - "成就2"
---
```

### 3. 添加图片

将图片放入 `source/assets/images/games/游戏名/` 目录，然后在 `photos` 中引用。

### 4. 生成网站

```bash
hexo generate
hexo server
```

## 现有示例

| 文件 | 轨道 | 说明 |
|------|------|------|
| elden-ring.md | single | 单机游戏示例（多图相册） |
| it-takes-two.md | multi | 联机游戏示例（好友、瞬间标签） |

## 字段说明

### 核心字段（必填）

- `game_id`: 游戏唯一标识，用于星图连线
- `track`: `single`（单机）或 `multi`（联机）
- `game_type`: 游戏类型
- `platform`: 游戏平台

### 单机区特有

- `completion_rate`: 完成度 %
- `achievements`: 成就列表

### 联机区特有

- `partners`: 好友列表
- `game_moments`: 瞬间标签（搞笑操作、纪念通关等）

### 媒体字段

- `photos`: 图片列表（多张自动成相册）
- `videos`: 视频列表
